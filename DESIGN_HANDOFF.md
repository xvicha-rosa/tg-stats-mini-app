# StatsFlow — Хендофф для дизайнера

Telegram Mini App для анализа соцсетей (TikTok / Instagram / YouTube).
Весь интерфейс — один файл `public/index.html` (HTML + CSS + JS внутри).
Дизайнер меняет **только стили и разметку**, логику (JS) не трогаем — она в конце файла в `<script>`.

## Технические ограничения

- **Платформа:** Telegram Mini App → открывается внутри Telegram, вертикальный экран
- **Ширина:** проектировать под **375px** (мобила). Тестовые девайсы 360–430px
- **Низ экрана:** фиксированный таб-бар 70px + safe-area (iPhone home indicator). Контент не должен под него залезать
- **Верх:** учитывать шапку Telegram (кнопка закрытия/назад сверху)
- **Шрифты Cyrillic:** текущий шрифт `AR_ZOZH` рендерит кириллицу крупным «капс»-стилем — если не нравится, дизайнер выбирает другой

# Экраны и состояния

Приложение = 4 вкладки (таб-бар снизу) + 2 модалки.

## 1. HOME (вкладка «Home»)
Маркетинговый экран, статика.
- Лого + название «StatsFlow» + подзаголовок «Социальный аналитический движок»
- Слоган (3 строки)
- Блок «Как это работает?» — 4 пронумерованных пункта
- Блок «Тарифы» — Бесплатно / Premium
- Блок «Почему StatsFlow» — 4 пункта с иконками

## 2. АНАЛИЗ (вкладка «Анализ»)
Главный экран. Имеет **3 состояния**, идущих сверху вниз:

**2a. Ввод (всегда видно)**
- Поле ввода ссылки (textarea) + кнопка «Анализировать →»

**2b. Результаты (появляются после анализа)**
- Карточка «Результаты»: 4 метрики в сетке 2×2:
  - Подписчики · Энгейджмент (%) · Лайки/просмотр (%) · Комментарии (%)
- Кнопка «← Новый анализ»
- Карточка «Что улучшить»: 3 совета (2 обычных + 1 инсайт со ✨)

**2c. Premium-блок (всегда видно под результатами)**
- Карточка «Premium подписка»: цена 50⭐ + список из 4 преимуществ + кнопка «Получить Premium →»

**2d. Premium-результат (после оплаты/списания, скрыт до этого)**
- Карточка «⭐ Premium анализ»:
  - Корень проблемы (выделенный блок)
  - Список инсайтов по категориям (Удержание, Алгоритм, Контент и т.д.)
  - Развёрнутый вывод (несколько абзацев)

## 3. УСЛУГИ (вкладка «Услуги»)
- Заголовок + описание
- Сетка услуг 2 колонки (8 карточек): иконка + название + срок + описание
- Подпись про цены внизу

## 4. ПРОФИЛЬ (вкладка «Профиль»)
- **Карточка профиля** (тёмный фон): аватар + имя + @username + бейдж статуса (🆓 Бесплатный / ⭐ N Premium-анализов)
- **Сетка статов 2×2**: «Анализов» (сделано) + «Доступно» (кредитов)
- Карточка «💰 Промокод» + кнопка «Применить промокод →»
- Карточка «Поддержка» + кнопка «Написать @xvichals»
- Карточка «О сервисе»

## МОДАЛКА 1 — Оплата Premium
Открывается по «Получить Premium →». Поверх затемнения.
- Заголовок «⭐ Получить Premium»
- Способ 1: «🎟 Списать 1 анализ» + остаток кредитов
- Способ 2: «⭐ 50 Telegram Stars» (пока неактивна, «Скоро»)
- Подсказка про промокод
- Состояние ошибки (нет кредитов)
- Кнопка «Закрыть»

## МОДАЛКА 2 — Промокод
Открывается по «Применить промокод →».
- Заголовок «💰 Промокод»
- Поле ввода кода + кнопка «Применить»
- Состояние успеха (зелёный блок: начислено N анализов)
- Состояние ошибки (красный блок: код не найден / уже использован)
- Кнопка «Закрыть»

# Как посмотреть состояния (для дизайнера)

Состояния с данными (результаты, premium, баланс) приходят с сервера и из Telegram —
в обычном браузере покажут пустые/нулевые значения и «Гость» вместо имени.
Чтобы увидеть наполненные экраны — открыть в Telegram через бота
[@tg_stats_mini_app_bot](https://t.me/tg_stats_mini_app_bot) и сделать анализ.

Промокод для теста Premium: `KUNISEKS1488` (даёт 1 бесплатный анализ).
